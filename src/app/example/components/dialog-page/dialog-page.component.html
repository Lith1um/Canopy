Dialogs

One of the repetitive and tedious jobs in Angular is to create dialogs. With the introduction of a simple yet powerful DialogService, it is easy to create customized dialogs on-the-fly. Various options can be configured, such as:

  • Dialog size
  • Auto height
  • Maximise window
  • Header/Footer buttons
  • Custom content
